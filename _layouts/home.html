--- layout: archive --- Sitio web que ofrece apoyo para dejar de fumar fumando una selecta mezcla de distintas hierbas.

Tres fases.
Descisión. Decidir una fecha para dejar de fumar, preparar el entorno para la fecha escogida y adquirir la mezcla. Esta fase es la más difícil, en esta web encontrarás contenido que podría ayudar.
Sustitución. Sustituir el tabaco por la mezcla durante 3 a 7 días aproximadamente.
Apoyo. Dejar de usar la mezcla pero mantenerla cerca para usarla sólo en caso de necesitarlo y evitar recaídas.
Dejar el tabaco requiere tres fases claras. Primero viene la decisión: elegir una fecha concreta, preparar el entorno eliminando todo lo relacionado con el tabaco y conseguir la mezcla sustitutiva. Esta fase inicial es crucial y la más difícil psicológicamente. Luego está la fase de sustitución, donde durante 3 a 7 días se reemplaza completamente el tabaco por la mezcla alternativa, rompiendo así el hábito físico. Finalmente llega la fase de autonomía, donde se deja de usar la mezcla pero se mantiene disponible para momentos de crisis, evitando así recaídas mientras se consolida la nueva vida libre de humo.
