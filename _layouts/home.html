--- layout: archive --- Sitio web que ofrece apoyo para dejar de fumarcon una selecta mezcla de distintas hierbas.

En resumen el proceso se podría dividir en tres etapas:

Fase de descisión, información, elegir una fecha para dejar de fumar, preparar el entorno para la fecha escogida y conseguir la mezcla sustitutiva. Esta fase es la más difícil, en esta web encontrarás contenido que podría ayudar. Fase de sustitución, reemplazo completo el tabaco por la mezcla durante 3 a 7 días aproximadamente. Apoyo, dejar de usar la mezcla pero mantenerla disponible para momentos de crisis y evitar así recaídas.
